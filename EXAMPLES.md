# 📘 Usage Examples - CSS Skeleton Loaders

These examples show how to use skeleton loaders in real-world UI scenarios.

---

## 🔹 Basic Card Loader

```html
<div class="skeleton-card">
  <div class="skeleton skeleton-title"></div>
  <div class="skeleton skeleton-text"></div>
  <div class="skeleton skeleton-text short"></div>
</div>
```

---

## 🔹 Profile Loader

```html
<div class="profile-skeleton">
  <div class="skeleton avatar"></div>
  <div class="info">
    <div class="skeleton skeleton-text"></div>
    <div class="skeleton skeleton-text short"></div>
  </div>
</div>
```

---

## 🔹 List Loader

```html
<ul>
  <li class="skeleton skeleton-text"></li>
  <li class="skeleton skeleton-text"></li>
  <li class="skeleton skeleton-text short"></li>
</ul>
```

---

## 🔹 Image Placeholder

```html
<div class="skeleton skeleton-image"></div>
```

---

## 🔹 Button Loader

```html
<button class="skeleton skeleton-button"></button>
```

---

## 💡 Tip

Use skeleton loaders while:
- Fetching API data
- Loading images
- Rendering dynamic UI

This improves perceived performance and user experience 🚀
